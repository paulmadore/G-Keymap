Importing special action plugins

A special action is a plugin which can be executed if a key is pressed. You can import new special action plugins, by going to the "Preferences" tab. Now you can see all currently available plugins installed on your system. To import a new plugin click on the "Import..." button. After doing this the new plugin will appear in the list of plugins.

Click the "Download..." button to download extra plugins from the keyTouch website.
