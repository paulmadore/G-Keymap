Keyboard

This part shows the currently used keyboard. You can change the current keyboard by clicking on the "Change..." button. As a result keytouch will be closed and the keytouch-keyboard program will be started to let you choose the keyboard. After choosing the keyboard in keytouch-keyboard, keytouch will be restarted. For more information about the keytouch keyboard program see section 3.5 and 4.1.
