About the keyTouch development

KeyTouch is developed with a lot of care. In these days where it is allowed that software contains bugs, I strive for developing a program without bugs and spend a lot of time make this come true. So if you discover a bug in keyTouch (it doesn't matter if it is a little one) do not hesitate to report it and I hope you can forgive me. You can report bugs at the SourceForge.net project page http://sourceforge.net/projects/keytouch/.
