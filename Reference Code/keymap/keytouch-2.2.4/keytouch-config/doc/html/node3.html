About this manual

This manual describes how to use keyTouch 2.2. I (the author of this manual and the developer of keyTouch) think it is very important that software is well documented. Therefor you can always contact me to improve this manual or to notice poor English.

This manual is written in LATEX to make it easy to modify and convert it to many different formats.
