Setting your preferences

Because you want to work as comfortable as possible, keyTouch allows you to set your preferences. If you want to do this go to the "Preferences" tab. Here you can set your "Documents" directory, the URL of your home- and search page, internet browser, e-mail- and chat program. These values will be used by plugins. The "Preferences" tab also contains a list of special action plugins. This will be thread in the next paragraph.
