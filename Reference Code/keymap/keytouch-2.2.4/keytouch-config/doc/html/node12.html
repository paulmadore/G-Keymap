The keytouch-keyboard user interface

This section discusses the user interface of the keytouch-keyboard program. The "Current keyboard" frame contains the name of the current keyboard. To change the current keyboard you select a keyboard in the list and press the "OK" button. If you don't want to change the current keyboard press the "Cancel" button.
