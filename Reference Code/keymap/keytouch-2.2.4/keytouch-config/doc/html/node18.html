Saving your settings

If your want to save the changes you made, you can click the "OK" or the "Apply" button at the bottom of the main window. If you click the "Apply" button, only the changes will be saved. If you click the "OK" button, the changes will be saved and keyTouch will be closed. You can also discard the changes and close keyTouch by clicking the "Cancel" button.
