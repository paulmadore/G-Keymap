Selecting your keyboard

The first thing you want to do is selecting the keyboard you use. To do this go to the "Keyboard" tab and click the "Change..." button. Because you need root privileges to change the keyboard, you will be asked for the root password. After that a window will appear. This window contains a list with all available keyboards. Select your keyboard from the list and click the "OK" button.

After choosing the keyboard, keytouch will be restarted and the key list will contain the extra function keys of your keyboard.

If your keyboard is not in the list, check the keyTouch website if there is a keyboard file available for your keyboard. For list of available keyboard files go to http://keytouch.sourceforge.net/dl-keyboards.html. Click the "Import..." button after downloading the correct keyboard file. A window will appear where you can select the keyboard file. After this your keyboard will be visible in the list of available keyboards.
